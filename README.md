This is a bot to book classes on the AimHarder platform automatically.

## Getting Started

To run the code locally, you'll need to have Python and [Poetry](https://python-poetry.org/) installed. Follow these instructions to get set up.

### Prerequisites

Ensure you have Python 3.11 and Poetry installed on your machine.

- Install Poetry:

    You can install Poetry with the following command:

    ```bash
    curl -sSL https://install.python-poetry.org | python3 -
    ```

- Verify the installation:

    ```bash
    poetry --version
    ```

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/hribera/aimharder-book-bot.git
    cd aimharder-book-bot
    ```

2. Install dependencies using Poetry:

    ```bash
    poetry install
    ```

3. Activate the virtual environment:

    ```bash
    poetry shell
    ```
